Others (unofficial)

  • Neper2Rhino: import Neper tessellation into Rhino and Grasshopper.

  • Phon: insert interface elements and export to Abaqus or OOFEM. Note: interface elements can be inserted directly in Neper using -interface cohesive.

  • Neper2CAE and PyCiGen: generate a columnar grain-growth polycrystal, insert interface elements and export to Abaqus (hex elements)
